签名算法

  1. 将请求参数(排除sign参数)按Ascii码序进行排序并拼接成字符串(具体签名字段,请看文档)。
  2. 将发放的apiSecret拼接在最后 &key=apiSecret。
  3. 将拼接后的字符串进行MD5加密, 转成小写。
示例
需要请求的参数和得到的apiSecret

apiSecret="abcdefg"

{
  "timestamp": 1583308458233,
  "id": "userId",
  "username": "username",
  "nickname": "nickname",
  "age": "18"
}

接口中要求参加签名的字段为：timestamp，username，id

拼接请求参数

"id=userId&timestamp=1583308458233&username=username"

拼接apiSecret

"id=userId&username=username&key=abcdefg"

进行MD5加密

"3126ceb292bdfcfae2f63e602282b081"
