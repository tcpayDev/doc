##Signature algorithm

  1. Sort the request parameters(excluding the sign parameter) according to the Ascii code sequence and splice them into string(during the splicing process, null values “”(empty) values and values that are not in string format are not required to be signed)

  2. Splice the apiSecret at the end

  3. Sha256 encrypt the concatenated string and convert to lower case.

#####Example ######Need to request parameters and get apiSecret apiSecret="abcdefg"

{
  "timestamp": 1583308458233,
  "id": "userId",
  "username": "username",
  "nickname": null,
  "age": ""
}

######Splice request parameters "id=userId&username=username"

Be care:

  • age is empty string，so don’t sign it.
  • nickname value is null，so don’t sign it.
  • timestamp value is not string, so don’t sign it
