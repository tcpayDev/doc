主动查询

GET /diamond/html/query

参数名参数类型示例值参数含义是否签名签名前是否加密(默认为否)
tradeNoStringno_1547108859849接入方生成的订单号, 要确保唯一性是
apiKeyStringtradingchain接入方申请的apiKey是
userIdString接入方用户id是
operSysTypeString用户操作系统类型安卓:Android, 苹果:ios是
timeStampString155185409535213位时间戳是
signStringc34b649d7de35b2e516efcf298ae063a签名值-

请求示例

GET http://192.168.124.154:8080/diamond/html/query?tradeNo=no_1552013313978&apiKey=tradingchain&userId=123456&operSysType=Android&timeStamp=1552013205012&&sign=c34b649d7de35b2e516efcf298ae063a
