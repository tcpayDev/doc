模式2 - 聊天+币商列表

请求方式

POST请求，Content-Type为application/json

请求URL

/mode2/sell

请求字段

字段类型长度示例可否为空说明是否签名
apiKey字符串-"tc"否商务后台创建的apiKey(亲联系您的上级代理,获取该标识,以及对应的签名key)是
tradeNo字符串16-32"12345678"否商户单号(16-32位,任意字符)是
amount字符串-"100"否交易金额(单位:元),不能小于等于0是
payNotifyUrl字符串-"1"否异步通知url,该url请不要携带特殊符号,请正常填写如"http://****/order/notify"是
sign字符串-"1"否签名否

返回内容

字段类型示例说明
code整型100000请求后台返回操作状态，100000为成功操作
msg字符串"OK"对code进行说明
url字符串"http://localhost:8092/mode2/buy_in_list?apiKey=tc-admin&tradeNo=DEV-DEMO-1577334350879&amount=1&list=%5B%7B%22accountName%22%3A%22MIGO"可以直接访问的链接,GET请求

{

    "tradeNo":"test-202002030200002",
    "amount":"1",
    "payNotifyUrl":"localhost",
    "receiveType":"UNIONPAY",
    "sign":"fca46f2f80e32402e29bbd19f88037b69ec29f0882b86c49f1d01075b04c3395"
}
