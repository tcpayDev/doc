模式5 - 交易所-授权接口

请求方式

POST请求，Content-Type为application/json

请求URL

/mode5/getLoginInfo

请求字段

字段类型长度示例可否为空说明是否签名
apiKey字符串-"tc"否商务后台创建的apiKey(亲联系您的上级代理,获取该标识,以及对应的签名key)是
payNotifyUrl字符串-"1"否异步通知url,该url请不要携带特殊符号,请正常填写如"http://****/order/notify"是
balanceUrl字符串-"http://payway***.com"，用户平台方可提现资产数量否签名是
userId字符串-"uh21ehu13eh1"否用户id，长度2-12位是
sign字符串-"1"否签名否

返回内容

字段类型示例说明
code整型100000请求后台返回操作状态，100000为成功操作
msg字符串"OK"对code进行说明
username字符串"1245648978477"授权标识-用户名
password字符串"998154152121156156"授权标识-密码

请求示例：

{
    "apiKey":"tc-admin",
    "payNotifyUrl":"localhost",
    "balanceUrl":"http://payway***.com",
    "userId":"uh21ehu13eh1",
    "sign":"d3302c87d3b1080cbb7b6ab23f5cb19e3c6db853af39717d826547bf01991384"
}

返回示例：

{
    "code": 100000,
    "msg": "OK",
    "userName": "1245648978477",
    "password": "998154152121156156"
}
